Say Hello! to TeXtended.

TeXtended is an all new LaTeX Editor written for OS X in Objective C. It’s free and beta, so feel free to test the app and contact us for feature requests and issue reports. Below is a list of features, the latest release can be downloaded here

Classic Features

  • syntax highlighting
  • multiple compilers
  • error and message console
  • search function
  • OS X autocorrection with LaTeX support
  • live preview
  • variable fonts and font sizes
  • goto line command
  • line bookmarks
  • autocompletion for commands and environments
  • retina support
  • tabs and multi window support
  • BibTex integration
  • full screen support
  • multi screen support
  • and much more …

Some Highlights

  • flexible single document and tex project architecture
  • awesome customizable and intelligent autocompletion
  • support for TODO comments in source code
  • structure outline view for chapters, sections, …
  • messages from your LaTex compiler, lacheck and chktex
  • extended information and hints for compiler warnings and errors
  • templates for documents and projects
  • customizable editor colors
  • drag’n’drop include for files (customizable)
  • count words, symbols, rows and sentences in the resulting PDF file
  • multiple main documents for a tex file (customizable for projects and documents)
  • TeXdoc integration for package information
  • autocompletion for cite commands (customizable)
  • DBLP integration for cite commands
  • ultra adaptable compiler architecture (you are able to use shell scripts as compilers or use our build in compile scripts)
  • translucent backpage for pdf fine tuning
  • base line support in pdf view
  • export as single file (paper publication)
  • and much more …
